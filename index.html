Portfolio


Here's a small insight of what I have designed. Some of the apps layout designed here are in Production and can be checked out live too!



Promptia

Promptia uses speech-to-text technology to figure out the portion of text you have spoken. It then highlights the portion of the text that you have spoken, and scrolls down accordingly, thus giving much better control than other teleprompter apps. Still in beta, this app is being published to Google Play Store, and was built using Flutter.

Personal Website

Had to come up with a flexible design for my personal website. I wanted a design which could be easily updated from my website and won't screw up the UI and UX much. I built this website using React and Firebase. The website can easily be updated from a mobile app thus giving me full ease and control on editing data on the website without multiple deployments.



Polydian

Polydian is a smart web app that takes the URL of a website as an input, and then proceeds to summarise the website and process it in a database. Built on MERN, this project uses url summariser as a package amongst other things to summarise the URL and process it into a Mongo DB. The summary of the website can then be exported as TXT files.
